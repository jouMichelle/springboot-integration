# springboot整体基础代码

说明：

> 此工程确定目录结构配置文件结构

| 目录       | 说明               |
| ---------- | ------------------ |
| cache      | 缓存工具实现存放   |
| common     | 公共常亮，枚举类   |
| config     | 配置类与自定义配置 |
| exception  | 全局异常           |
| service    | 服务实现方法       |
| controller | 对外提供接口       |
| utils      | 工具类存放         |
| starter    | 启动类存放         |
